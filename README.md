## How to use

### Aliases

Use `symlink` to link to the `.zshrc` in here rather than in your root directory.
e.g. `ln -s ~/code/devx/.zshrc ~/.zshrc`

### Cursor settings

`ln -s ~/code/devx/cursor/settings.json ~/Library/Application\ Support/Cursor/User/settings.json`

### Requirements

Requires jq
`brew install jq`
